# 바크로 월별 매출 보고서 — 기획안 v2.0

## 0. 한 줄 요약
`sales.csv` (매출 raw) + `target.csv` (월별 목표)를 입력으로 받아 **매월 임원 보고용 다각도 분석 대시보드**를 생성하는 Next.js + ECharts + shadcn 로컬 웹 앱. URL 쿼리(`?month=2026-04`)로 보고월을 바꿔 동일 규격을 매달 재사용. **2026-04를 기준월**로 첫 빌드.

v1 대비 v2 주요 차이:
- 9개 탭 (종합·목표 달성·수출·B2B·B2C·면세점·브랜드 분석·변동 분석·인사이트)
- target.csv 통합으로 본월/분기 목표 달성률을 핵심지표 카드와 매트릭스에 노출
- 모든 라벨 한국어 (영어 약자 제거), 비교는 절대값+차이금액+변화율 동시 표시
- "왜 늘었나/줄었나" 변화 요인 워터폴 차트 + 신규/이탈 거래처 + 신제품 효과 분석

## 1. 기술 스택과 실행 방식

| 영역 | 선택 | 이유 |
|---|---|---|
| 프레임워크 | **Next.js 15 (App Router)** | Server Component로 CSV 집계를 요청 시점에 끝내고, 클라이언트는 차트만 렌더 → 159K행도 1초 이내 |
| 차트 | **Apache ECharts + echarts-for-react** | 워터폴/트리맵/히트맵/게이지 등 풍부, 한국어 툴팁/폰트, Apache 2.0 무료 |
| UI | **shadcn/ui (Radix) + Tailwind** | 카드/탭/배지/셀렉트/툴팁 폴리시 |
| 표 | **TanStack Table** | 정렬/검색/페이지/sticky 헤더 |
| 데이터 처리 | Node 단에서 **PapaParse + 자체 집계** | 외부 의존 최소, sales.csv를 한 번 파싱 후 모든 화면에서 공유 |
| 실행 | `npm run dev` → http://localhost:3000 | 매달: `sales.csv` + `target.csv` 교체 → 새로고침 |
| 스냅샷 | `npm run snapshot -- 2026-04` → 단일 HTML | 임원 회람용 |

## 2. 데이터 파이프라인

### 2.1 원본 컬럼 (17개)
`채널, 날짜, 주문번호, 제품명, 품목코드, 판매수량, 실 매출, 주문금액, 할인금액, 수수료, 배송비, 정산금액, 딜러, 거래처, 거래처 사업형태, 원가, 브랜드`

### 2.2 정제 규칙

| 이슈 | 처리 |
|---|---|
| 날짜 포맷 혼재 (`2023-12-21`, `2026. 4. 9`) | 정규식으로 분리 → ISO 변환 |
| 숫자 콤마/공백 (`11,400`) | 콤마 제거 후 Number |
| 원가 `#N/A` | NaN 처리, 마진 KPI에서 제외하고 데이터 품질 배지로 비율 노출 |
| 실매출 0 행 | **매출 집계에서 제외**, 별도 "비매출 출고" 카드/탭으로 분리 노출 |
| 주문번호 단위 집계 | 주문건수(unique 주문번호), AOV(주문당 평균) 계산 |
| 빈 딜러 (203행) | "미지정"으로 통일 |

### 2.3 파생 컬럼 (앱 내부에서만)

- `category`: `수출` | `B2B` | `B2C` | `면세점`
- `subCategory`: 수출→국가, B2B→거래처유형, B2C→채널그룹
- `brandHouse`: `자체` (레노덤·레노덤프로·헤이우) | `수입` (네오·엑스비·크리스티나) | `기타`
- `channelGroup`: `공식몰` | `종합몰` | `소호몰` | `면세점` | `B2B` | `수출` | `임직원/패밀리`
- `gp`(이익) = `실매출 - 원가 × 판매수량` (원가 NaN인 행은 NaN)
- `isNonRevenue`: 실매출 0 또는 사업형태가 증정/임직원/마케팅용/파손/테스트인 경우 true

## 3. 분류 매핑 (사용자 정의 명시화)

매핑 파일(`config/mappings.ts`)에 **모두 상수**로 두고 매월 점검.

### 3.1 채널 → 대분류

- **수출**: `수출`
- **B2B**: `B2B몰`
- **면세점**: `면세점` *(B2C 안의 별도 그룹이지만 KPI는 독립)*
- **B2C**: 위 셋을 제외한 모든 채널 (16종)

### 3.2 B2C 내 채널 그룹

| 그룹 | 채널 |
|---|---|
| 자사 공식몰 | 레노덤 공식몰, 레노덤 스마트스토어, 엑스비앙스 공식몰, 엑스비앙스 스마트스토어, 헤이우 공식몰, 바크로하우스, 바크로하우스 스마트스토어 |
| 종합몰 | W컨셉, SSG, 쿠팡, 쿠팡 로켓, 쿠팡 그로스, 큐텐, 화해 |
| 소호몰 | 소호몰 |
| 임직원/패밀리 | 바크로패밀리, 헤메코랩 |

### 3.3 브랜드 → 하우스

- **자체**: 레노덤, 레노덤 프로페셔널, 헤이우
- **수입**: 네오스트라타, 엑스비앙스, 크리스티나
- **기타**: 기타 *(증정/리플렛/도구류 — 매출 집계에서 보통 0)*

### 3.4 브랜드 ↔ 공식몰 매핑 (브랜드별 공식몰 추이용)

- 레노덤: `레노덤 공식몰` + `레노덤 스마트스토어`
- 엑스비앙스: `엑스비앙스 공식몰` + `엑스비앙스 스마트스토어`
- 헤이우: `헤이우 공식몰`
- 바크로하우스(다브랜드): `바크로하우스` + `바크로하우스 스마트스토어`

### 3.5 수출 국가 (거래처 사업형태에서 추출)

`해외 (XXX)` 또는 `수출 (XXX)` 패턴에서 괄호 안만 추출. 미매칭(`해외 (기타)`, `수출 (기타)`)은 **"기타"**로.

### 3.6 비매출 출고 (사업형태)

증정 (기타), 증정 (마케팅), 증정 (영업), 임직원, 직원, 거래처 직원, 마케팅용, 테스트 (수입허가), 파손제품, 교육 → **비매출 출고**로 묶어 별도 노출.

## 5. 페이지별 상세 기획

### Tab 1. 종합 — "이번 달, 한 페이지로"

**1) 헤더 KPI 4개 카드** — 모두 MoM·QoQ·YoY ▲▼% 동시 표기

- 총 실매출
- 총 주문건수 (unique 주문번호)
- 평균 주문금액(AOV)
- 총 판매수량

**2) 대분류 도넛 + 막대 콤보**

- 좌: 수출/B2B/B2C/면세점 4분류 도넛
- 우: 같은 4분류의 12개월 스택바 추이

**3) 기준월 일별 매출 라인** — 토/일·공휴일 음영 + 큰 스파이크 일자에 데이터 포인트 라벨

**4) Top 10 거래처 / Top 10 제품** — 2개 테이블 (실매출/수량/MoM%)

**5) 비매출 출고 카드** — 증정/임직원/마케팅용/파손 등으로 나간 "매출 0" 출고를 별도로 가시화 (수량·원가 합)

**6) 데이터 품질 배지** — 원가 #N/A 비율, 미매칭 채널/브랜드 경고

---

### Tab 2. 수출 (해외영업팀)

**1) KPI 카드**: 수출 총매출, 거래국가 수, Top 거래처

**2) 국가별 매출 가로막대** — MoM·YoY 변화 화살표 동시 표기

**3) 국가별 12개월 추이 라인** — 다선 차트, 큰 비중 국가 강조

**4) 국가 × 브랜드 히트맵** — 어느 시장에 어떤 브랜드가 깔리는지

**5) 거래처별 표** — 거래처, 국가, 매출, 수량, 전월비

---

### Tab 3. B2B (국내영업팀)

**1) KPI 카드**: B2B 총매출, 활성 거래처 수, 평균 거래처당 매출

**2) 거래처 유형별 도넛 + 추이** — 병원 / 피부관리실 / 대리점 / 프랜차이즈 / 기타

**3) 영업사원별 막대 (실적 보드)** — 본월 실적 + MoM·YoY 화살표 + 분기 누적
- **본월 매출 0인 사원은 표시 제외** (사용자 정책)

**4) 영업사원 × 거래처유형 매트릭스**

**5) 신규/이탈 거래처** — 본월 첫 매출 거래처, 직전 6개월 대비 매출 0이 된 거래처 (영업 액션포인트)

**6) 브랜드별 B2B 매출 비중** — 자체 vs 수입

### Tab 4. B2C (브랜드별)

> "면세점 제외 B2C"가 디폴트. 토글로 면세점 포함도 가능.

**1) KPI 카드**: B2C 총매출, 자체몰 비중, 종합몰 비중, 소호몰 비중

**2) 브랜드별 매출 (가로막대)** — 자체 vs 수입 색 구분

**3) 브랜드별 채널그룹 분해 스택바** — 레노덤이 공식몰/종합몰/소호몰 어디서 얼마나 났나

**4) 브랜드별 공식몰 12개월 추이 (소형 멀티)**

- 엑스비앙스 공식몰+스마트스토어

- 바크로하우스(자사몰)
→ 라인 4개 + 지난해 동월 점선

**5) 종합몰 채널별 표** — W컨셉/SSG/쿠팡 등의 매출, 수량, MoM, 수수료율, 정산매출

**6) Top 제품 (B2C 전체)**

**1) KPI 카드**: 면세 총매출, MoM·YoY, 거래처 수

**2) 12개월 추이 라인 + 전년 동월 점선**

**3) 면세 거래처별 매출**

**4) 면세 브랜드 분해**

**5) 일별 출고 라인** — spiky 패턴 가시화

### Tab 6. 인사이트

> 자동 통계 카드 + **사람이 작성한 코멘트** 결합. 매월 `insights/YYYY-MM.md` 파일을 추가하면 페이지 상단에 마크다운 그대로 렌더.

**자동 카드들**:
1. 본월 베스트/워스트 카테고리 — 전월 대비 매출 ±상위 3
2. 신제품 추적 — 본월 처음 매출 발생한 SKU (직전 13개월 무매출 → 본월 ≥1)
3. 이탈 위험 SKU — 직전 3개월 평균 대비 본월 -50% 이상
4. 요일별 매출 패턴
5. 거래처 집중도 (Top10 비중, HHI지수)
6. 브랜드 × 채널그룹 히트맵
7. 할인율 = 할인금액/주문금액 — 채널그룹별 평균
8. 수수료율 = (주문금액-정산금액-할인금액)/주문금액
9. (부가) 마진율 — 원가 #N/A 제외, 채널그룹별 GP율 (참고 라벨)

**사람 코멘트**:
- `insights/2026-04.md` 같은 파일에 마크다운으로 자유 작성
- 페이지 상단에 "이번 달 코멘트" 박스로 노출

## 6. KPI/계산식 정의

| 지표 | 정의 |
|---|---|
| 실매출 | `실 매출` 컬럼 합 |
| 주문건수 | unique `주문번호` |
| AOV | 실매출 / 주문건수 |
| 수량 | `판매수량` 합 |
| 정산매출 | `정산금액` 합 |
| 수수료율 | `수수료` / `주문금액` |
| 할인율 | `할인금액` / `주문금액` |
| GP | `실매출 - 원가 × 수량` (원가 NaN 행 제외) |
| GP율 | GP / 해당 부분집합 실매출 |
| MoM | (당월-전월) / 전월 |
| QoQ | (당분기 누적-직전분기 누적) / 직전분기 누적 |
| YoY | (당월-전년동월) / 전년동월 |

분기 진행 중일 땐 카드 위에 "분기 진행 중 (N/3개월)" 표시.

## 7. 비교 로직과 시각 규약

- 양수 = **녹색 ▲**, 음수 = **빨강 ▼**, ±2% 이내 = **회색 ●**
- 전년 동월(YoY) 데이터가 없는 항목은 "신규/N.A."로 표기
- 차트의 점선/그림자는 항상 "전년 동월" 의미로 통일
- 통화 단위는 **백만원(M)** 기본, 카드 hover 시 정확한 원 단위 노출

## 8. 매월 운영 워크플로우

1. **데이터 교체**: `sales.csv`를 최신 파일로 덮어쓰기
2. **자동 검증**: `npm run check` → 매핑 누락 채널/브랜드/사업형태 경고
3. **인사이트 추가**: `insights/2026-05.md` 생성, 본월 코멘트 작성
4. **보고서 출력**: 브라우저에서 `?month=2026-05` → 인쇄 또는 `npm run snapshot -- 2026-05`

새 채널/거래처가 등장해도 코드 수정 없이 자동 분류:

- 채널: 화이트리스트 매핑 → 미매칭은 자동으로 "기타 B2C"로 떨어지되 경고
- 거래처: 패턴(`해외 (XXX)`, `병원`, `피부관리실`) 우선

## 11. 정책 결정 (사용자 확정)
